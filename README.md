
Random Number Generator (React)

A simple Random Number Generator application built using React. This project demonstrates how to manage state, handle button events, and use conditional rendering to update the UI dynamically.

🚀 **Features**

Generates a random number within a specified range (e.g., 1–100)
Updates the displayed number instantly on button click
Displays a placeholder message before generating a number
Uses conditional rendering for dynamic UI changes
Clean and simple user interface

🛠️ **Technologies Used**
React.js
JavaScript (ES6)
HTML
CSS

💡** Overview**

This application uses a React functional component with the useState hook to store and update the generated random number. When the user clicks the button, a new number is generated and displayed instantly. Conditional rendering is used to show a default message before any number is generated.
